## System.out.println();

syso를 타이핑후 컨트롤+ 스페이스를 눌러보세요.

이클립스에서 컨트롤 + 스페이스는 자동완성기능을 가지고 있습니다.

```java
public class TestController {

	public void start() {
		syso  // 여기서 Control + Space 누르세요.
	}

}
```

### 결과

```java
public class TestController {

	public void start() {
		System.out.println();  // 여기서 Control + Space 누르세요.
	}

}
```
## 라인 삭제 (Ctrl + d)

Ctrl + d 누르시면 커서가 있는 라인이 삭제됩니다.

또한 여러줄의 라인이 드래그된 상태에서 단축키를 눌러도 삭제됩니다.

```java
public class TestController {

	public void start() {
		System.out.println();  // 이 라인에 커서를 두고 Control + D
	}

}
```

## 자동 정렬(Ctrl + Shift + f )

줄맞춤을 할때 라인이 딱딱 정렬이 되있어야 소스보기 편하고 유지보수할때도 쉽습니다.

```java
public class TestController {

	public void start() {
		// System 라인들이 굉장히 지저분 합니다.
			System.out.println();
		System.out.println();
		   System.out.println();
	System.out.println();
	System.out.println();
	}

}
```

### 결과

```java
public class TestController {

	public void start() {
		// System 라인들이 정렬되어 이쁘네요.
		System.out.println();
		System.out.println();
		System.out.println();
		System.out.println();
		System.out.println();
	}

}
```
