```sql
USE Database;
 
SELECT 'GRANT SELECT, UPDATE, INSERT, DELETE ON "' + TABLE_SCHEMA + '"."' + TABLE_NAME + '" TO "designer"' AS query
FROM
	information_schema.tables
ORDER BY
	query;
```

* 원하는 권한 (SELECT, UPDATE, INSERT, DELETE)에 맞게 사용
